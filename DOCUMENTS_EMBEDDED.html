Guide d'utilisation

📋 Comment ajouter un document

Allez dans l'onglet Ajouter et renseignez les métadonnées :

  • Titre clair et précis (ex: Attestation bancaire Afriland 2026)
  • Catégorie (juridique, banque, fiscalité, RH…)
  • Niveau de confidentialité adapté
  • Date d'expiration si applicable (très important pour les rappels)
  • Notes complémentaires (numéro de référence, contact, etc.)

Le fichier lui-même sera uploadé dans une prochaine version via un canal sécurisé.

📄 Différence entre métadonnées et fichier

Métadonnées = informations sur le document : titre, catégorie, date d'expiration, statut. Stockées dans la base de données veraluz_documents.

Fichier = le document lui-même (PDF, scan, image). Stocké dans un bucket Supabase Storage privé. Jamais accessible publiquement. Accessible uniquement via URL signée de courte durée (15 min max).

🔐 Niveaux de confidentialité

  • public — Information non sensible, peut être partagée librement
  • internal — Usage interne uniquement — pas de partage externe
  • confidential — Données sensibles — accès restreint aux managers et propriétaire
  • restricted — Très sensible — accès propriétaire uniquement (contrats bancaires, titres, NIF…)

🏦 Pourquoi les documents bancaires et RH restent privés

Les documents bancaires (relevés, conventions, IBAN) et RH (contrats, bulletins) contiennent des informations personnelles et financières protégées. Ils doivent rester dans des buckets privés et ne jamais être accessibles via une URL publique.

  • Bucket veraluz-bank-private → documents bancaires
  • Bucket veraluz-hr-private → contrats et dossiers employés
  • Bucket veraluz-legal-private → statuts, actes, NIF
  • Bucket veraluz-payslips-private → bulletins de paie (future paie formelle)

🤖 Comment l'agent juridique utilisera les métadonnées

L'agent legal_docs_ai (statut : draft) pourra dans le futur :

  • Détecter les documents manquants dans la checklist
  • Alerter sur les documents expirés ou proches de l'expiration
  • Préparer des questions pour le notaire, le fiscaliste ou la banque
  • Créer des recommandations à valider par le propriétaire

Ce qu'il ne fera pas : lire le contenu des fichiers, donner des avis juridiques définitifs, signer ou envoyer des documents, remplacer un avocat ou notaire.

⚙️ Ce qui n'est pas encore automatisé

  • Upload de fichiers (PROMPT 020+)
  • Notifications automatiques d'expiration par email/SMS
  • Génération automatique de bulletins de paie
  • Export bancaire de documents
  • Activation de l'agent juridique (PROMPT 021+)
  • Partage sécurisé externe avec lien signé
