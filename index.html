👥 Partager avec un collaborateur

Pour qu'un collègue (technicien à Kribi, par exemple) accède aux mêmes données : envoyez-lui simplement l'URL de cette page. Dès qu'il l'ouvre, il voit exactement les mêmes données que vous et peut ajouter/modifier en direct.

⚠️ Gardez l'URL privée — toute personne qui l'obtient peut lire et modifier vos données.

💾 Backup JSON (optionnel)

Télécharge un fichier de sauvegarde avec toutes tes données. Utile pour archiver ou pour restaurer après un accident.

🔗 Héberger sur GitHub Pages (optionnel)

Pour accéder à l'app depuis n'importe quel appareil avec une URL publique :

  1. Connectez-vous à github.com
  2. Créez un dépôt : mako-city-chantier
  3. Uploadez ce fichier sous le nom index.html
  4. Settings → Pages → Branch : main
  5. URL : https://VOTRE-USERNAME.github.io/mako-city-chantier
